Ejercicios - Desarrollo web

Práctica de HTML

Construir un archivo HTML usando las etiquetas principalmente conocidas: p, div, table, span.

Se sugiere tomar un tema de su agrado, por ejemplo si es aficionado a algún equipo deportivo. Puede construir una página con el nombre, la descripción, imagenes, etc.

Puede ver un ejemplo de lo esperado en el siguiente link: Ejercicio 1

Práctica de CSS

Tomar el HTML creado en el ejercicio anterior y ahora, mejorar su visual, aplicando estilos CSS.

Debe aplicar el uso de los selectores estudiados en el curso, propiedades para mejorar colores, fuentes y posicionamiento. En el caso de posicionamiento debe usar FlexBox y GridCSS.

Puede ver un ejemplo de lo esperado en el siguiente link: Ejercicio 2

SASS - SCSS

Aplicar SASS-SCSS al archivo de estilos del ejercicio anterior, de forma que colores, tamaños, modelos de caja (margenes y rellenos), posiciones y cualquier otra propiedad usada, sea posible definirla con variables, mixings y herencia por lo menos.

Puede ver un ejemplo de lo esperado en el siguiente link: Ejercicio 3

BEM - Media Queries

Llegó la hora de dejar nuestro proyecto usable de forma responsiva, además de ajustar los nombres de las clases CSS usadas.

Ajustar usando Media Queries para que tu proyecto se vea correctamente en cualquier dispositivo.

Puede ver un ejemplo de lo esperado en el siguiente link: Ejercicio 4

Git - GitHub

Es momento de publicar tu proyecto.

Debes crear una cuenta (si no la tienes) en github.com y subir a la nube todo lo realizado usando git.

Puede acceder al proyecto de ejemplo publicado en el siguiente link: Ejercicio 5
